Como fazer as requests?

As requests devem ser feita com o método POST em /api/predict enviando um multipart/form-data.

  • O formato de arquivo deve ser image/jpeg ou image/png
  • O nome do field da imagem DEVE SER image
  • Não pode exceder 10MB
